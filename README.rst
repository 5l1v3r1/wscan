wscan
=====

A Fast & Simple web site scanner.
Base on aiohttp and refer to the dirsearch of multi-threading version.
Can both run in Linux & Windows.

Features
-------
- Fuzz web site path
- Mapping a site map
- Multi-co-routine
- User-agent randomization
- Custom extensions
- Custom maximum of co-routine
- Friendly interface
